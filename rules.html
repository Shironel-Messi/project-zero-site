PROJECT ZERO Rules

以下のルールはお互いリスペクトを持って対戦をするため、過去行われたトーナメントルールを改良したものです。
このサーバー間（以下：身内）および他者とのトラブルを回避するためにFOBをされている方は絶対読んでください。

●両者超重要事項

  • rat以外の相手に対し、初手からいきなり行動の選択を与えない行為（ガンハメ）は厳禁です。 特にポイント目当てのガンハメ行為だった場合、このサーバー参加間のトラブルでも基本重い処罰になります。
  • rat以外のプレイヤーへの煽り行為は原則禁止です。

＊rat…以下のルールに違反する者、または違反することを良しとする者のこと

●防衛側

  1. 以下の武器およびキャラクター等の使用は原則禁止です
    オートワームホール生成器（awh） / エナジーウォール（ew） / クワイエット（桑） / フルトンランチャー / 貫通銃
    デバイスグリッチ（ターボコントローラーなど）
  2. 弾切れや入れ違い等による故意的な自殺は原則禁止です
  3. アンブッシュ（whを第１甲板以外に設置して起動する行為）は原則禁止です
    アンブッシュ…ワームホール生成器（wh）を用いて裏どり、または急接近する行為
  4. フレンドpvpにおける殺傷および回収は避けましょう。
    攻守交代で対戦している際、自分が潜入側で相手にチャンスを貰った場合、 それ以降の試合はフレンドpvpに含まれる可能性があります。
  5. サーバー内特別ルール
    相手プレイヤーが身内または親しいフレンドで「ゴールは絶対しない」と宣言した場合、 非殺傷砂を使用しましょう。不慮のヘッドショットによる瞬殺を防ぐためです。

●潜入側

  1. 以下の武器およびキャラクター等の使用は原則禁止です
    オートワームホール生成器（awh） / エナジーウォール（ew） / クワイエット（桑） / フルトンランチャー / 貫通銃
    デバイスグリッチ（ターボコントローラーなど）
  2. 防衛者乱入前に第１・２甲板の兵士に手を出した瞬間、ゴールは原則禁止です。
    ゴールなし条件で対戦することは可能です。

    電源設備、迫撃砲、ドローン破壊の際に意図せず警備兵に当たった場合や、 1～3人の第２甲板の警備兵が第３甲板に入ってきてやむを得ず排除した場合は該当しません。
  3. フレンドpvpおよび正規防衛者からチャンスを貰った場合のゴールは原則禁止です。

    身内同士の対人は基本フレンドpvpに含まれ、ゴールはノーチャンスでもNGです。 core vs coreなどゴール有条件で行う場合は、事前に相手へ伝え了承を得てください。 野良相手のノーチャンスゴールについては各自判断に任せます。
  4. チャンス扱いについて
    明らかに射線が通っているのに肩や足など、ヘッドショットを避けたラインに １発だけ撃たれるのもチャンス扱いです。

    チャンスか判断できない場合でも、 「明らかにここで負けていた」と思える場面があるならゴールしない方が望ましいです。
  5. 身内に対してのフルトングレネードは事前許可を。
    原則使用して良い相手はratのみです。 フレンドpvpの際は相手の許可を事前に得てから使用しましょう。

●まとめ

相手がratであり、その証拠を押さえることができれば 煽りメッセージ以外のルールを守る必要はありません。
